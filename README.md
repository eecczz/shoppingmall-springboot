# 🛒 쇼핑몰 웹 서비스 (Spring Boot 기반)

Java Spring Boot 기반의 서버사이드 쇼핑몰 웹 서비스입니다.  
회원가입부터 상품 조회, 장바구니, 결제(PortOne 테스트 결제)까지  
**쇼핑몰의 핵심 구매 흐름을 end-to-end로 구현**하였습니다.

본 프로젝트는 **Spring MVC + JPA + QueryDSL + MySQL** 기반으로 설계되었으며,  
**jcloud 환경에 배포**하여 실제 서버에서 동작하도록 구성했습니다.

---

## 🔗 배포 환경

우분투 기반 학교 클라우드 **jcloud** 환경(Ubuntu)에 배포되어 있으며, 외부에서 아래 주소로 접속 가능합니다.
- **외부 접속 URL:** `http://113.198.66.75:10134/demo/list`
- **Cloud**: jcloud
- **Server**: Spring Boot 단일 서버 배포
- **DB**: MySQL (운영 DB)
- **결제 연동**: PortOne (테스트 결제 모드)

> ※ 실제 결제는 테스트 결제만 구현되어 있으며, 실 결제는 연동하지 않았습니다.

---

## 📌 주요 기능

### 1️⃣ 회원 기능
- 회원가입 / 로그인 (서버 세션 기반)
- 로그인 상태에 따른 화면 분기
- 로그아웃 처리

### 2️⃣ 상품 관리
- 초기 실행 시 **청크 데이터 방식으로 상품 임의 등록**
- 상품 목록 조회
- QueryDSL 기반 검색 및 페이징 처리

### 3️⃣ 장바구니
- 상품 장바구니 추가
- 장바구니 수량 변경
- 장바구니 상품 삭제
- 장바구니 상품 개수 실시간 표시

### 4️⃣ 주문 / 결제
- 장바구니 기반 주문 생성
- **PortOne 결제 테스트 연동**
- 결제 검증(imp_uid 기반)
- 결제 완료 시 장바구니 비우기 처리

---

## 🔄 전체 서비스 흐름

1. 회원가입 후 로그인
2. 상품 목록 조회
3. 상품 상세 페이지 확인
4. 장바구니 상품 추가
5. 장바구니 페이지에서 수량 조절
6. 결제 페이지 이동
7. PortOne 테스트 결제 진행
8. 결제 성공 → 주문 완료

## 🧱 기술 스택

### Backend
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- QueryDSL
- Spring Transaction

### Database
- MySQL
- JPA 기반 엔티티 설계
- 연관관계 매핑 및 양방향 관리

### Payment
- PortOne 결제 API (테스트 모드)

- jcloud 서버 배포
- 단일 Spring Boot 애플리케이션 실행

---

## 🗂️ DB 설계 개요

- **Member** : 회원 정보
- **Post(Product)** : 상품 정보
- **Cart** : 회원별 장바구니
- **SavedItem** : 장바구니-상품 매핑 엔티티
- **Order** : 주문 정보

> JPA 연관관계 기반으로 설계하였으며,  
> 조회 성능 개선을 위해 QueryDSL을 활용한 동적 쿼리를 적용했습니다.
